# ⚡ БЫСТРЫЙ ЗАПУСК

## 🎯 Два способа запуска:

---

## Способ 1: Через BAT файлы (ПРОЩЕ!)

### 1. Запустите сервер
Двойной клик на файл: **`start-server.bat`**

Должно появиться окно с текстом:
```
🚀 Сервер запущен на порту 5001
```

### 2. Запустите клиент
Двойной клик на файл: **`start-client.bat`**

Должно появиться окно и откроется браузер.

---

## Способ 2: Через терминал

### Терминал 1 (Сервер):
```bash
cd server
node server-simple.js
```

### Терминал 2 (Клиент):
```bash
cd client
npm run dev
```

---

## ✅ Что должно быть:

### Окно сервера:
```
🚀 Сервер запущен на порту 5001
✅ Работает в режиме in-memory (без MongoDB)
⚠️  Данные будут потеряны при перезапуске сервера
```

### Окно клиента:
```
VITE v5.x.x  ready in xxx ms

➜  Local:   http://localhost:5173/
➜  Network: use --host to expose
```

### Браузер:
Автоматически откроется `http://localhost:5173`

## 🐛 Если не работает:

### "node не является внутренней командой"
**Решение:** Установите Node.js с https://nodejs.org

### "Cannot find module"
**Решение:** 
```bash
cd server
npm install
```

### Порт 5001 занят
**Решение:** Закройте другие приложения или измените порт в `server-simple.js`
