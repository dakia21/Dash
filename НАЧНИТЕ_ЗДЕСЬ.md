## ⚡ Быстрый старт (3 минуты)

### 1. Запустите сервер
```bash
cd server
node server-simple.js
```

### 2. Запустите клиент (новый терминал)
```bash
cd client
npm run dev
```

### 3. Откройте браузер
```
http://localhost:5173
```

**Готово!** Зарегистрируйтесь и начните общаться.

## 🆘 Нужна помощь?

### Проблемы с запуском?
1. Убедитесь, что Node.js установлен: `node --version`
2. Установите зависимости: `npm install` в папках client и server
3. Проверьте, что порты 5001 и 5173 свободны

### Проблемы с деплоем?
1. Проверьте **[ДЕПЛОЙ.md](ДЕПЛОЙ.md)** - раздел "Проблемы?"
2. Проверьте **[DEPLOYMENT_GUIDE.md](DEPLOYMENT_GUIDE.md)** - раздел "Решение проблем"

### Другие вопросы?
- Создайте issue на GitHub
- Проверьте консоль браузера (F12)
- Проверьте логи сервера в терминале
