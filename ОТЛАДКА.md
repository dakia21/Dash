## 🚀 Как тестировать с логами

### 1. Откройте консоль браузера
- Нажмите **F12**
- Перейдите на вкладку **Console**
- Оставьте консоль открытой

### 2. Тест фото

**Действия:**
1. Нажмите на скрепку 📎
2. Нажмите "Фото"
3. Выберите изображение
4. Нажмите "Отправить"

**Что должно быть в консоли:**
```
📷 Клик на "Фото"
📁 handleFileSelect вызван
   Файл: image.jpg image/jpeg 123456 bytes
✅ Файл выбран: image.jpg
🖼️ Создан URL для превью
📤 Начинаем отправку сообщения
   chatId: 1234567890
   текст: 
   файл: image.jpg
📎 Конвертируем файл в base64...
✅ Файл сконвертирован в base64
🖼️ Тип: изображение
🌐 Отправляем на сервер...
✅ Сообщение создано на сервере: 1234567891
🔄 Отправляем через WebSocket
```

**Если что-то не так:**
- Если нет "📷 Клик на Фото" - кнопка не работает
- Если нет "📁 handleFileSelect" - input не срабатывает
- Если нет "✅ Файл сконвертирован" - ошибка чтения файла
- Если нет "✅ Сообщение создано" - ошибка сервера

## 🐛 Типичные ошибки и решения

### Ошибка 1: "❌ Не могу отправить: нет текста/файла или chatId"
**Причина**: Чат не выбран или нет контента  
**Решение**: Убедитесь что вы открыли чат

### Ошибка 2: "❌ Файл не выбран"
**Причина**: Вы отменили выбор файла  
**Решение**: Попробуйте снова и выберите файл

### Ошибка 3: "❌ Ошибка чтения файла"
**Причина**: Файл поврежден или слишком большой  
**Решение**: Попробуйте другой файл (меньше 10MB)

### Ошибка 4: "Доступ к микрофону запрещен"
**Причина**: Браузер не имеет разрешения  
**Решение**: 
- Chrome: Настройки → Конфиденциальность → Настройки сайта → Микрофон
- Разрешите доступ для localhost

### Ошибка 5: "Микрофон не найден"
**Причина**: Микрофон не подключен  
**Решение**: Подключите микрофон или наушники с микрофоном

### Ошибка 6: "📦 Размер аудио: 0 bytes"
**Причина**: Микрофон не записывает  
**Решение**: 
- Проверьте что микрофон работает в других приложениях
- Попробуйте другой браузер
- Перезагрузите страницу

### Ошибка 7: "❌ Ошибка отправки: ..."
**Причина**: Ошибка на сервере  
**Решение**: 
- Проверьте что сервер запущен
- Посмотрите логи сервера
- Проверьте консоль браузера на детали ошибки

## 📊 Проверка сервера

### Логи сервера должны показывать:

**При отправке сообщения:**
```
POST /api/messages 201
```

**При ошибке:**
```
POST /api/messages 500
Error: ...
```

### Если сервер не отвечает:
1. Проверьте что сервер запущен: `cd server && node server-simple.js`
2. Проверьте порт: должен быть 5001
3. Проверьте что нет ошибок в консоли сервера

## 🔍 Дополнительная отладка

### Если фото не отображается:
1. Откройте консоль (F12)
2. Перейдите на вкладку Network
3. Найдите запрос POST /api/messages
4. Посмотрите Request Payload - там должен быть base64
5. Посмотрите Response - там должно быть сообщение с fileUrl

### Если голосовое не воспроизводится:
1. Проверьте что в сообщении есть fileUrl
2. Попробуйте открыть fileUrl в новой вкладке
3. Должно начаться скачивание или воспроизведение

### Если WebSocket не работает:
1. Откройте консоль (F12)
2. Должно быть "✅ Подключено к серверу"
3. Если нет - проверьте что сервер запущен
4. Перезагрузите страницу
