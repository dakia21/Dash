## 🧪 Быстрый тест (5 минут)

### 1. Запустите проект

**Терминал 1:**
```bash
cd server
node server-simple.js
```

**Терминал 2:**
```bash
cd client
npm run dev
```

---

### 2. Тест фото (1 минута)

1. Откройте чат
2. Нажмите скрепку 📎
3. Выберите "Фото"
4. Выберите любое изображение
5. Нажмите "Отправить"

**✅ Ожидаемый результат:**
- Фото отображается в чате
- Собеседник видит фото

---

### 3. Тест голосового (1 минута)

1. Убедитесь что поле ввода пустое
2. Кнопка должна быть зеленой 🎤
3. Нажмите на микрофон
4. Разрешите доступ к микрофону (если спросит)
5. Говорите 2-3 секунды
6. Нажмите "Отправить"
7. Нажмите Play на голосовом сообщении

**✅ Ожидаемый результат:**
- Голосовое сообщение появилось
- При нажатии Play воспроизводится звук

---

### 4. Тест файла (1 минута)

1. Нажмите скрепку 📎
2. Выберите "Файл"
3. Выберите любой файл (PDF, DOC и т.д.)
4. Нажмите "Отправить"
5. Нажмите на файл в чате

**✅ Ожидаемый результат:**
- Файл отображается с иконкой
- При клике файл скачивается

## 🐛 Если не работает

### Голосовое не записывается
- Дайте разрешение на микрофон в браузере
- Проверьте что микрофон работает

### Фото не отображается
- Попробуйте файл меньше 5MB
- Проверьте консоль браузера (F12)
