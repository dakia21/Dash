## 🐛 Если что-то не работает

### Сообщения не доходят
**Проверьте:**
1. Консоль браузера (F12) - есть ли ошибки?
2. Терминал сервера - есть ли логи WebSocket?
3. Оба пользователя в одном чате?

**Решение:**
```bash
# Перезапустите сервер
cd server
node server-simple.js
```

### Голосовые сообщения не работают
**Проверьте:**
1. Разрешен ли доступ к микрофону?
2. Есть ли микрофон на компьютере?
3. Консоль браузера - есть ли ошибки?

**Решение:**
- Разрешите доступ к микрофону в настройках браузера
- Используйте HTTPS (для production)

### Изображения не загружаются
**Проверьте:**
1. Размер файла (макс 50MB для изображений)
2. Формат файла (JPG, PNG, GIF)
3. Консоль браузера - есть ли ошибки?

**Решение:**
- Используйте изображения меньшего размера
- Проверьте формат файла
