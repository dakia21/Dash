# ⚡ Быстрые команды

## 🚀 Локальный запуск

### Запустить сервер:
```bash
cd server
node server-simple.js
```

### Запустить клиент:
```bash
cd client
npm run dev
```

### Открыть в браузере:
```
http://localhost:5173
```

---

## 📦 Установка зависимостей

### Все сразу:
```bash
npm run install-all
```

### Только клиент:
```bash
cd client
npm install
```

### Только сервер:
```bash
cd server
npm install
```

## 🔍 Отладка

### Проверить порты:
```bash
# Windows
netstat -ano | findstr :5001
netstat -ano | findstr :5173

# Linux/Mac
lsof -i :5001
lsof -i :5173
```

### Убить процесс на порту (Windows):
```bash
# Найти PID
netstat -ano | findstr :5001

# Убить процесс (замените PID)
taskkill /PID <PID> /F
```

### Убить процесс на порту (Linux/Mac):
```bash
kill -9 $(lsof -t -i:5001)
kill -9 $(lsof -t -i:5173)
```

## 📝 Переменные окружения

### Локально (client/.env.local):
```env
VITE_API_URL=http://localhost:5001/api
VITE_SOCKET_URL=http://localhost:5001
```

### Production (client/.env.production):
```env
VITE_API_URL=https://your-backend.onrender.com/api
VITE_SOCKET_URL=https://your-backend.onrender.com
```

### Сервер (на Render.com):
```env
NODE_ENV=production
JWT_SECRET=your_secret_key_here
PORT=5001
```
