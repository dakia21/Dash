## 🚀 НАЧНИТЕ ПРЯМО СЕЙЧАС:

### ШАГ 1: Откройте приложение
Перейдите по ссылке: **http://localhost:5173**

Вы увидите красивую страницу входа в стиле Telegram! 🎨

### ШАГ 2: Настройте MongoDB (чтобы регистрация работала)

**Самый быстрый способ - MongoDB Atlas (облачная, бесплатная):**

1. **Зарегистрируйтесь**: https://www.mongodb.com/cloud/atlas/register
2. **Создайте кластер**: FREE tier (M0) - бесплатно навсегда
3. **Настройте доступ**:
   - Database Access → Создайте пользователя
   - Network Access → Add IP → `0.0.0.0/0`
4. **Получите строку подключения**: Connect → Connect your application
5. **Обновите `server/.env`**:
   ```env
   MONGODB_URI=mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/messenger
   ```
6. **Сервер автоматически перезапустится!**

📖 **Подробная инструкция**: См. файл `SETUP_MONGODB.md`

### ШАГ 3: Зарегистрируйтесь и начните общаться!
1. Создайте аккаунт
2. Войдите в систему
3. Найдите пользователей через поиск
4. Отправляйте сообщения в реальном времени! ⚡
