## 🔧 Техническая информация:

### Запущенные процессы:
- **Terminal 5**: Frontend (React + Vite) - http://localhost:5173
- **Terminal 7**: Backend (Node.js) - http://localhost:5000

### Архитектура:
- **Frontend**: React 18 + TypeScript + Socket.IO Client
- **Backend**: Node.js + Express + Socket.IO + In-Memory Storage
- **Аутентификация**: JWT токены + bcrypt
- **Реальное время**: WebSocket через Socket.IO

### Особенности:
- **In-Memory хранилище**: Данные в оперативной памяти
- **Быстрая работа**: Нет задержек с базой данных
- **Полная функциональность**: Все возможности мессенджера
- ⚠️ **Данные временные**: При перезапуске сервера данные пропадут
