# 🚀 КАК ЗАПУСТИТЬ СЕРВЕР

## Шаг 1: Откройте терминал

### Windows:
- Нажмите `Win + R`
- Введите `cmd` или `powershell`
- Нажмите Enter

### Или в VS Code:
- Нажмите `Ctrl + ~` (тильда)
- Или: View → Terminal

---

## Шаг 2: Перейдите в папку server

```bash
cd server
```

---

## Шаг 3: Запустите сервер

```bash
node server-simple.js
```

---

## ✅ Что вы должны увидеть:

```
🚀 Сервер запущен на порту 5001
✅ Работает в режиме in-memory (без MongoDB)
⚠️  Данные будут потеряны при перезапуске сервера
```

---

## 🎯 Если сервер уже запущен:

1. Нажмите `Ctrl + C` чтобы остановить
2. Запустите снова: `node server-simple.js`

---

## 🐛 Если ошибка "Cannot find module":

```bash
npm install
```

Затем снова:
```bash
node server-simple.js
```

---

## 📋 Полная последовательность команд:

```bash
# 1. Перейти в папку server
cd server

# 2. Если нужно - установить зависимости
npm install

# 3. Запустить сервер
node server-simple.js
```

---

## ✅ Сервер запущен! Что дальше?

### Откройте ВТОРОЙ терминал для клиента:

```bash
# В новом терминале:
cd client
npm run dev
```

---

## 🔍 Теперь тестируйте:

1. Сервер работает на `http://localhost:5001`
2. Клиент работает на `http://localhost:5173`
3. Откройте браузер: `http://localhost:5173`
4. Откройте консоль браузера (F12)
5. Попробуйте отправить фото
6. Смотрите логи в ОБЕИХ консолях!

---

**ВАЖНО:** Держите оба терминала открытыми!
- Терминал 1: Сервер (логи сервера)
- Терминал 2: Клиент
- Браузер F12: Логи клиента
