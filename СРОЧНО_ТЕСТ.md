## 🧪 БЫСТРЫЙ ТЕСТ (2 минуты)

### 1. Запустите проект

```bash
# Терминал 1
cd server
node server-simple.js

# Терминал 2
cd client
npm run dev
```

### 2. Откройте консоль браузера
- Нажмите **F12**
- Вкладка **Console**

### 3. Попробуйте отправить фото

1. Откройте чат
2. Нажмите скрепку 📎
3. Нажмите "Фото"
4. Выберите изображение
5. Нажмите "Отправить"

### 4. СМОТРИТЕ В КОНСОЛЬ!

**Должно быть:**
```
📷 Клик на "Фото"
📁 handleFileSelect вызван
✅ Файл выбран: image.jpg
📤 Начинаем отправку сообщения
📎 Конвертируем файл в base64...
✅ Файл сконвертирован в base64
🖼️ Тип: изображение
🌐 Отправляем на сервер...
✅ Сообщение создано на сервере
```

**Если что-то не так - ПОКАЖИТЕ МНЕ ЧТО В КОНСОЛИ!**

## 🎤 Тест голосового

1. Поле ввода пустое
2. Нажмите зеленую кнопку 🎤
3. Говорите 2-3 секунды
4. Нажмите "Отправить"

## 🐛 Если не работает

### ПОКАЖИТЕ МНЕ:
1. **Что написано в консоли браузера (F12)**
2. **Что написано в консоли сервера**
3. **Какая именно ошибка**

### Типичные проблемы:

**"❌ Не могу отправить: нет chatId"**
→ Чат не выбран, откройте чат

**"❌ Файл не выбран"**
→ Вы отменили выбор, попробуйте снова

**"Доступ к микрофону запрещен"**
→ Разрешите доступ в настройках браузера

**"Микрофон не найден"**
→ Подключите микрофон
