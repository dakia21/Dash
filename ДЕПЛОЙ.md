# 🚀 Быстрый деплой мессенджера

## 📦 Что нужно сделать:

### 1️⃣ Загрузить код на GitHub (5 минут)

**Вариант А: Автоматически (Windows)**
```bash
deploy.bat
```

**Вариант Б: Вручную**
```bash
# Инициализируйте git
git init

# Добавьте файлы
git add .
git commit -m "Initial commit"

# Создайте репозиторий на GitHub.com и выполните:
git remote add origin https://github.com/ВАШ_USERNAME/ВАШ_REPO.git
git branch -M main
git push -u origin main
```

### 2️⃣ Настроить GitHub Pages (2 минуты)

1. Откройте ваш репозиторий на GitHub
2. Перейдите в **Settings** → **Pages**
3. В разделе **Source** выберите **GitHub Actions**
4. Готово! Сайт будет доступен через 2-3 минуты

**Ваш сайт:** `https://ВАШ_USERNAME.github.io/ВАШ_REPO/`

### 3️⃣ Задеплоить сервер на Render.com (5 минут)

1. **Зарегистрируйтесь:** [render.com](https://render.com) (можно через GitHub)

2. **Создайте Web Service:**
   - Нажмите **New +** → **Web Service**
   - Подключите ваш GitHub репозиторий
   - Выберите репозиторий

3. **Настройте:**
   ```
   Name: messenger-backend
   Root Directory: server
   Build Command: npm install
   Start Command: node server-simple.js
   Instance Type: Free
   ```

4. **Добавьте переменные окружения:**
   ```
   NODE_ENV = production
   JWT_SECRET = любая_длинная_случайная_строка
   PORT = 5001
   ```

5. **Нажмите Create Web Service**

6. **Скопируйте URL** (например: `https://messenger-backend-xxxx.onrender.com`)

### 4️⃣ Обновить URL сервера (2 минуты)

1. Откройте файл `client/.env.production`

2. Замените URL на ваш:
   ```env
   VITE_API_URL=https://ваш-сервер.onrender.com/api
   VITE_SOCKET_URL=https://ваш-сервер.onrender.com
   ```

3. Откройте `server/server-simple.js` и обновите CORS (строка ~10):
   ```javascript
   const io = socketIo(server, {
     cors: {
       origin: [
         "http://localhost:5173",
         "https://ВАШ_USERNAME.github.io"
       ],
       methods: ["GET", "POST"]
     }
   });
   ```

4. Сохраните и загрузите изменения:
   ```bash
   git add .
   git commit -m "Update production URLs"
   git push
   ```

### 5️⃣ Готово! 🎉

Подождите 2-3 минуты и откройте:
```
https://ВАШ_USERNAME.github.io/ВАШ_REPO/
```

## 🐛 Проблемы?

### Сайт не открывается
- Подождите 2-3 минуты после push
- Проверьте вкладку **Actions** на GitHub - должна быть зеленая галочка
- Проверьте Settings → Pages - должен быть указан URL

### Не подключается к серверу
- Убедитесь, что сервер запущен на Render (зеленый статус)
- Проверьте URL в `client/.env.production`
- Проверьте CORS в `server/server-simple.js`

### Сервер "засыпает"
- Это нормально для бесплатного плана Render
- Первый запрос после 15 минут неактивности будет медленным (~30 сек)
- Для постоянной работы нужен платный план ($7/месяц)
