## Как запустить проект

### Сервер
```bash
cd server
npm install
node server-simple.js
```
Сервер запустится на `http://localhost:5001`

### Клиент
```bash
cd client
npm install
npm run dev
```
Клиент запустится на `http://localhost:5173`

## Технологии

### Frontend
- React 18 + TypeScript
- Vite
- Zustand (state management)
- Socket.IO Client
- Axios
- React Router
- date-fns
- Lucide React (иконки)

### Backend
- Node.js + Express
- Socket.IO
- JWT (jsonwebtoken)
- bcryptjs
- In-memory хранилище (Map)

---

## Известные особенности

1. **In-memory хранилище**: Данные хранятся в памяти и теряются при перезапуске сервера
2. **Без MongoDB**: Используется временное решение с Map вместо базы данных
3. **Порты**: Сервер - 5001, Клиент - 5173
