### Frontend:
- **Framework:** React 18 + TypeScript
- **Build:** Vite
- **State:** Zustand
- **HTTP:** Axios
- **WebSocket:** Socket.io Client

### Backend:

- **Framework:** Express

- **Auth:** JWT + bcrypt

### Деплой:
- **Frontend:** GitHub Pages (бесплатно)

## 🔥 Ключевые изменения в коде

### server/server-simple.js (строка ~380):
```javascript
// БЫЛО:
io.to(data.chatId).emit('receive-message', data);

// СТАЛО:
socket.to(data.chatId).emit('receive-message', data);
```
**Эффект:** Сообщения теперь отправляются всем КРОМЕ отправителя

### client/src/api/axios.ts:
```javascript
const API_URL = import.meta.env.VITE_API_URL || 
                (import.meta.env.DEV 
                  ? 'http://localhost:5001/api' 
                  : 'https://your-backend-url.com/api')
```
**Эффект:** Автоматическое определение URL для dev/production

### client/src/hooks/useSocket.ts:
```javascript
const SOCKET_URL = import.meta.env.VITE_SOCKET_URL || 
                   (import.meta.env.DEV 
                     ? 'http://localhost:5001' 
                     : 'https://your-backend-url.com')
```
**Эффект:** Автоматическое определение WebSocket URL
